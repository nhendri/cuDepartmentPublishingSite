## Site Conventions
Namespaces and naming conventions are used to denote any configuration or code based customizations that extend or build on top of the OOTB features and pages.

#### Namespaces and Naming Conventions

| Prefix | Scope                         | Used In                                                                                                                            |

| cu_    | template level customizations | <ul style="list-style-type: none; padding-left: 0;"><li>Site Columns</li><li>Content Types</li><li>Page Layout Templates</li></ul> |
| HR_    | site level customizations     | <ul style="list-style-type: none; padding-left: 0;"><li>Site Columns</li><li>Content Types</li></ul>                               |
